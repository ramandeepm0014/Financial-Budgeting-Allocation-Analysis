# 💰 Financial Budget Allocation Dashboard
An interactive Business Intelligence Dashboard built using Streamlit, Pandas, and Plotly for analyzing financial budget allocation, utilization, revenue trends, and departmental performance.


## 🚀 Features

### 📊 Financial Overview
- Total Budget Allocated
- Budget Utilized
- Remaining Budget
- Revenue Generated
- Allocation Efficiency

### 🎛 Interactive Filters
- Fiscal Quarter Filter
- Department Filter
- Expense Category Filter
- Budget Status Filter

### 📈 Advanced Visualizations
- Budget Efficiency Gauge Chart
- Budget vs Revenue Comparison
- Sunburst Chart
- Treemap Visualization
- Sankey Diagram
- Quarterly Budget Trend
- Budget Status Distribution
- Department-wise Budget Allocation
- Budget vs Utilization Scatter Plot
- Heatmap Analysis

### 📥 Download Center
- Download Filtered Dataset as CSV

### 📋 Data Exploration
- Interactive Data Table
- Dynamic Filtering
- Real-time Dashboard Updates


## 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- NumPy

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/financial-budget-allocation-dashboard.git
```

### Move into Project Folder

```bash
cd financial-budget-allocation-dashboard
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
